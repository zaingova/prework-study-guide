Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • You can style all elements in the same group by separating them with a comma in CSS.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: adds all local files to repository.
  • git commit -m "insert commit message here": records changes to the repository.
  • git pull origin main: pull changes from the locally stored branch origin/master and merge that to the local checked-out branch.
  • git push origin (insert branch name): push the current branch to the branch of the matching name in the remote repository.
